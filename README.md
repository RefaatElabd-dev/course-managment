# Course Management System

A simple full-stack Course Management app built with:

- ASP.NET Core (.NET 8)
- Angular 19
- Angular Material

## Features

✅ Course CRUD  
✅ Search, Pagination, Sorting  
✅ Fluent Validation  
✅ Soft Delete & Audit Trail  
✅ Angular Reactive Forms  
✅ Material UI  
✅ Global API Error Handling
